# MidManager-AI (中层管理者AI助手) 架构与设计方案

根据针对中层管理者的痛点调研，本项目旨在打造一个全天候的“数字化大秘”，帮助中层释放精力、破除协同壁垒、赋能团队发展并提升个人领导力。以下为项目核心架构与设计。

---

## 1. 核心技术架构图

系统采用现代化的分层架构，包含客户端、后端服务、Agent核心引擎、RAG知识增强模块以及动态技能插件市场。

```mermaid
graph TD
    %% 客户端层
    subgraph Frontend [客户端与交互层]
        UI[Web/Desktop 仪表盘]
        Chat[多模态对话界面]
        PluginStore[技能市场与插件管理]
    end

    %% 后端服务层
    subgraph Backend [后端服务与网关层]
        API[API Gateway]
        Auth[统一认证与企业级RBAC]
        TaskQueue[异步任务与定时调度]
    end

    %% Agent核心引擎层
    subgraph AgentEngine [核心Agent引擎层]
        Orchestrator[意图识别与任务编排器]
        LLM[大语言模型网关层]
        Memory[长期/短期记忆机制]
        SkillExecutor[动态技能沙箱与执行器]
    end

    %% RAG 知识库增强层
    subgraph RAG [知识库增强 (RAG)]
        VectorDB[(企业级向量数据库)]
        DocParser[制度/周报/文档解析器]
        Retriever[混合检索与重排引擎]
    end

    %% 插件市场与外部集成
    subgraph PluginMarket [插件与技能生态]
        SkillRegistry[云端技能注册中心]
        AutoInstaller[技能自动安装与热更新]
        ThirdPartyAPI[HR系统/OA/Jira等第三方集成]
    end

    %% 连线关系
    UI --> API
    Chat --> API
    PluginStore --> API

    API --> Auth
    API --> TaskQueue
    API --> Orchestrator

    Orchestrator --> LLM
    Orchestrator --> Memory
    Orchestrator --> SkillExecutor
    Orchestrator --> Retriever

    Retriever --> VectorDB
    DocParser --> VectorDB

    SkillExecutor --> AutoInstaller
    AutoInstaller --> SkillRegistry
    SkillExecutor --> ThirdPartyAPI
```

## 2. 专为中层管理者设计的50个核心技能

基于中层管理者的四大核心挑战（团队、绩效、项目协同、向上管理），我们将技能分为五大类，共计50个。

### 一、 团队赋能与人才发展 (Team & Talent)
1. **1v1面谈大纲生成器**：根据员工近期表现与性格特点生成面谈提纲。
2. **员工流失风险预警**：基于周报情绪、打卡及沟通频率预测离职倾向。
3. **团队能力雷达图**：一键生成全员技能矩阵，发现团队短板。
4. **新人Onboarding计划**：自动生成30/60/90天定制化融入计划。
5. **招聘JD智能优化**：匹配业务真实诉求，自动生成更有吸引力的岗位描述。
6. **授权与任务分配建议**：根据任务难度和员工成熟度，推荐最佳负责人。
7. **导师匹配引擎**：为新人自动匹配最合适的跨部门或组内导师。
8. **团队建设方案策划**：根据预算、人数和近期团队状态推荐团建活动。
9. **培训需求智能诊断**：分析项目难点与缺陷报告，输出团队急需的培训方向。
10. **员工关怀提醒**：自动跟踪入职周年、生日及特殊事件并提供关怀话术。

### 二、 目标对齐与绩效管理 (Goals & Performance)
11. **OKR/KPI 拆解与对齐**：将公司战略级目标自动拆解为团队和个人的可执行指标。
12. **年度绩效面谈草稿生成**：汇总全年周报与产出，生成客观的绩效评价草稿。
13. **低绩效(PIP)改进计划制定**：生成合规且具有可操作性的绩效改进方案。
14. **绩效面谈冲突降级**：提供应对员工不服绩效评定时的标准话术与心理建设。
15. **薪酬调整模拟器**：在有限预算内，提供最优的团队调薪建议模型。
16. **目标达成偏差预警**：根据当前进度，提前预测无法达成的目标。
17. **个人贡献度归因分析**：在复杂项目中客观评估每个成员的具体贡献比例。
18. **季度/年度复盘报告生成**：提炼成功经验与失败教训，一键生成结构化复盘文档。
19. **优秀员工提名理由提炼**：为高绩效员工生成强有力的晋升/表彰推荐信。
20. **战略偏离度检测**：定期检查团队日常任务与核心OKR的对齐情况。

### 三、 项目把控与敏捷协同 (Project & Execution)
21. **项目延期安抚话术**：生成向干系人解释延期原因并提供补救方案的沟通文案。
22. **跨时区协同排期优化**：为全球化团队计算最佳重合会议时间与异步沟通规范。
23. **资源瓶颈自动识别**：分析项目依赖链，提前指出可能卡脖子的资源环节。
24. **会议纪要与待办分发**：自动提取会议核心结论，生成并分发行动项(Action Items)。
25. **突发事件B-Plan生成**：针对核心节点，自动生成风险应对的备用预案。
26. **项目ROI与成本测算**：快速测算新需求的人力成本与预期收益。
27. **每日站会异常捕捉**：识别员工在站会汇报中的“隐藏风险”或“过度承诺”。
28. **供应商/外包进度跟进**：自动生成催促进度、核对交付物的标准化邮件。
29. **项目文档合规性审查**：检查交付物是否符合公司的架构和安全规范。
30. **敏捷迭代回顾优化**：分析Sprint数据，指出下个迭代需要改进的具体流程。

### 四、 向上管理与跨部门协同 (Upward & Cross-functional)
31. **向上汇报PPT大纲梳理**：根据老板的关注点（如重数据或重逻辑），生成汇报框架。
32. **跨部门资源申请话术**：分析对方利益诉求，生成互利共赢的资源请求邮件。
33. **背锅/推诿场景保护伞**：在权责不清时，生成不卑不亢、划清界限的回复文案。
34. **高管会议意图解读**：分析公司高管发言稿，提炼对本部门的潜在影响与要求。
35. **部门间冲突调解**：生成缓解跨部门矛盾、拉齐目标的破冰沟通方案。
36. **跨部门SLA(服务等级协议)辅助**：自动起草与其他部门合作的标准流程和响应时间。
37. **跨部门项目RACI划分**：自动生成谁负责、谁批准、谁咨询、谁知情的责任矩阵。
38. **争取预算的商业论证(Business Case)**：生成向高层要人、要钱的逻辑推演报告。
39. **向上风格适配器**：将同一份报告根据不同高管的偏好（激进型、稳健型）进行改写。
40. **业务线数据对齐检查**：检查提交给老板的数据是否与其他部门存在口径冲突。

### 五、 情绪价值与个人效能 (Emotional & Personal Effectiveness)
41. **管理者的情绪树洞**：提供安全、保密的压力倾听与情绪疏导对话通道。
42. **复杂邮件优先级排序**：智能识别哪些邮件需要立即处理，哪些可以延后或委派。
43. **“如何说不”的话术库**：委婉但坚定地拒绝不合理需求或过度塞任务的话术。
44. **职场政治与敏感度预警**：在发送敏感邮件或做出敏感决策前提供风险提示。
45. **碎片化时间微学习规划**：根据管理者日程，推荐每天15分钟的领导力提升内容。
46. **个人领导力盲区诊断**：通过团队反馈（360度评估）提炼个人的管理盲点。
47. **管理者“时间黑洞”分析**：分析日历，指出在低效会议或救火上浪费的时间比例。
48. **关键决策认知偏差提醒**：在做出重大决策前，检查是否存在幸存者偏差或确认偏误。
49. **职业发展与跳槽时机评估**：客观评估当前岗位的成长空间及外部市场价值。
50. **高难度对话心理建设**：在进行降薪、裁员、劝退前，进行AI角色扮演模拟演练。

## 3. “技能自动安装”机制的流程设计

为了让 Agent 保持轻量并在需要时具备强大的能力，系统设计了**动态感知与自动安装（Skill Auto-Installation）**机制。

### 核心流程步骤
1. **意图与技能缺口识别**
   - 用户输入请求（例：“帮我写一份针对张三的PIP改进计划”）。
   - Orchestrator（编排器）解析意图，发现本地未安装 `pip_plan_generator` 技能。
2. **云端市场匹配与检索**
   - Agent 自动向云端 `SkillRegistry` 发起请求，基于语义匹配找到最合适的官方或高赞第三方技能。
3. **安全与权限校验提示**
   - Agent 检查该技能所需的权限（如：读取HR系统数据）。
   - 若涉及敏感数据，通过 UI 向用户发起一次性授权请求（“该技能需要访问绩效数据，是否允许？”）。
4. **动态下载与沙箱加载**
   - 授权通过后，系统下载技能的 schema 和执行脚本。
   - 验证数字签名确保代码安全，随后将技能动态注入 `SkillExecutor` 沙箱环境（无需重启整个服务）。
5. **上下文注入与无缝执行**
   - Agent 将当前对话的上下文（如：张三的近期绩效、公司合规要求）注入该技能。
   - 技能执行并返回结果给用户。
6. **缓存与固化**
   - 技能被缓存在本地，下次遇到类似需求时实现毫秒级调用。用户也可在“我的插件”面板中对其进行版本锁定或卸载。

## 📖 项目简介
**MidManager-AI** 是一个专为企业**中层管理者**量身定制的开源 AI Agent 系统。
中层管理者常常夹在战略制定与基层执行之间，面临着“向上管理难、跨部门扯皮多、团队赋能累、绩效评估杂”等真实痛点。本项目旨在通过大语言模型（LLM）、私有化知识库（RAG）和动态技能生态，打造一个全天候的“数字化大秘”，帮助中层释放精力，专注于核心业务突破与领导力提升。

## ✨ 核心特性
- **🎯 专为中层设计的 50+ 技能矩阵**：内置向上汇报、跨部门资源申请、PIP计划制定、团队状态雷达图等专业能力。
- **🔌 动态“技能自动安装”**：对话中自动感知需求缺失，无缝从云端技能市场下载并加载所需能力，全程无需重启。
- **📚 企业级私有知识增强 (RAG)**：支持接入企业内部规章制度、历史周报、项目文档，让 AI 的建议合规且切中痛点。
- **🔒 绝对的数据隐私与安全**：针对绩效、薪酬等敏感数据，提供严格的沙箱隔离与本地化处理机制，确保数据不出域。

## 🛠️ 快速开始

### 前置要求
- Docker & Docker Compose
- OpenAI API Key 或本地大模型接口 (如 Ollama / vLLM)

### 一键部署
\`\`\`bash
# 1. 克隆仓库
git clone https://github.com/your-org/MidManager-AI.git
cd MidManager-AI

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入你的 LLM API Key 等信息

# 3. 启动所有服务
docker-compose up -d
\`\`\`
服务启动后，访问 \`http://localhost:3000\` 即可进入管理者控制台。
